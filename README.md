# KFZ-Pfretzschner.de

Statische Website für die KFZ-Werkstatt von Tim Pfretzschner.

## Projektbeschreibung

Diese Website präsentiert die Dienstleistungen und Informationen der KFZ-Werkstatt Tim Pfretzschner.

## Technologien

- Statische HTML/CSS/JavaScript
- Modernes, responsives Design
- SEO-optimiert

## Entwicklung

```bash
# Lokale Entwicklung
python3 -m http.server 8000
```

## Deployment

Die Website wird als statische Site gehostet.
